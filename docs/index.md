# Overview

Wiktionary to yomitan (wty) is a project to make Yomitan-compatible dictionaries from Wiktionary.

It also deals with this documentation page, and the online hosting of those dictionaries.

To get started, see how to [import](import.md) a wty dictionary to Yomitan, or the Yomitan [wiki](https://yomitan.wiki/getting-started/) for a comprehensive overview.

## Similar projects

- A frequency dictionary creator: [leipzig-to-yomitan](https://github.com/StefanVukovic99/leipzig-to-yomitan)
- For ebooks there is [ebook_dictionary_creator](https://github.com/Vuizur/ebook_dictionary_creator) that uses [pyglossary](https://github.com/ilius/pyglossary)
- [Rust engine](https://github.com/aecsocket/wordbase) that supports Yomitan.
